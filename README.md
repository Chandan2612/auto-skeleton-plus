# Auto Skeleton Plus

Automatically generate skeleton loaders from Flutter widgets using runtime + code generation.

---

## ✨ Features

- 🔄 Auto convert UI → skeleton
- ⚡ Runtime skeleton builder
- 🧠 Code generation support (`@GenerateSkeleton`)
- 🎯 Supports Text, Image, Column, Row
- 🚫 `SkeletonIgnore` support
- 🎨 Clean and lightweight
